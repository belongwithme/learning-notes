左侧菜单什么时候用？记住这张“任务 → 入口”表

⌂
Overview

确认当前生产版本、域名、提交和整体健康度。

↻
Deployments

查看每一次版本、构建状态、预览地址，或选择回滚候选。

≡
Logs

排查运行时错误；构建失败则进入具体 Deployment 查看 Build Logs。

◎
Domains

管理自定义域名、证书和 DNS 验证状态。

⚙
Settings

管理 Git、构建命令、Node 版本、环境变量与访问保护。

🧭
先判断对象，再点菜单

“项目怎么配置”去 Settings；“某次为什么失败”去 Deployments；“上线后请求为什么报错”去 Logs。对象分清，排查就不会在控制台里迷路。
